# Touhou-Meishuugeki-Cursed-Sweeper-and-Bewildering-Parallel
A cursed maid, caused by an unseen enemy.
## Info
***Touhou Meishuugeki ~ Cursed Sweeper and Bewildering Parallel*** is a *Touhou Project* fangame developed by Drillimation Systems in collaboration with E. Enthusiasm. The game centers on a mysterious substance created by the Gensokyo Socialite Order under the name of glohm, who plans to use it to sabotage an upcoming autumn equinox festival hosted by Queen Marisa in Kozankyo in the Federal Kingdom of Gensokyo. Unlike the previous two SNES-style fangames, the graphics are more detailed complete with CD-quality audio.

## How to Play
*Touhou Meishuugeki ~ Cursed Sweeper and Bewildering Parallel* can be played using a keyboard or controller. The goal is pretty much the same as in every other *Touhou* game: Defeat the stage boss to move onto the next stage.

- The arrow keys, left analog stick, and D-pad move the character.
- The Z key and A button shoots.
- The Shift key and B button (the L button does it too) focuses.
- The X key and X button (the R button does it too) uses a spell card.

## Tools used to make this game:
- GameMaker Studio (the game itself)
- Visual Studio Code (for programming cutscenes and other objects storing text)
- Paint.NET (for most of the game's sprites)
- Adobe Photoshop (for some of the complex graphical jobs)
- Clip Studio Paint (for posing characters)
- OpenMPT (for the game's soundtrack)
